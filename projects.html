Programming and Software: Tic Tac Toe with Python

Description

Completion of a Tic Tac Toe game where the user can play against a programmed AI, but the AI never loses! Source code via Github. (INCLUDE HYPERLINK)

Specifications

Achieved by AI search implementing a minimax algorithm to select the best move after computing utility value for every possible final board outcome. Use of pygames library for user interface.

Usage

> python3 runner.py

Mechanical Engineering Design Project: Flood Evacuation Device

Description

Engineering design project in a team of 3 engineering students focusing on accomodating for potential consequences of climate change and global warming. Practice of the engineering design process from problem definition to prototyping.

Specifications

  • Engineering based teamwork in a semi-profesional setting
  • Simulated project management as the leader of the group
  • Development of various engineering designs to solve a given problem
  • Implementation of concept development tools and techniques to progress through the design process, most notably:
    1. Pugh Matrix
    2. House of Quality
    3. Morph Chart
    4. Function-means tree
    5. Objectives tree
    6. Pairwise comparion chart
